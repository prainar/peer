## 🚀 Start Development in 3 Steps

### 1. Get the Code
```bash
git clone https://github.com/prainar/peer.git
cd peer
git checkout module-4
```

### 2. Start Backend
```bash
cd app/backend
python3 -m venv myenv
source myenv/bin/activate
pip install -r requirements.txt
python3 main.py
```

### 3. Start Frontend
```bash
cd app/frontend
npm install
npm run dev
```
